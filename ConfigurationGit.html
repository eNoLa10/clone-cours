Configuration de Git

Pourquoi configurer Git ?

La configuration de Git est essentielle pour identifier l'auteur de chaque modification (commit) et pour collaborer efficacement avec d'autres développeurs. Cela garantit un suivi clair de l'historique des contributions dans un projet.

Étapes pour configurer Git

  1. Ouvrir Git Bash : Lance Git Bash pour entrer les commandes de configuration.
  2. Configurer le nom d'utilisateur :
    Utilise la commande :
    git config --global user.name "Ton Nom"
    Remplace "Ton Nom" par ton nom complet.
  3. Configurer l'adresse e-mail :
    Utilise la commande :
    git config --global user.email "ton.email@example.com"
    Remplace "ton.email@example.com" par ton adresse e-mail.
  4. Vérifier la configuration :
    Pour vérifier tes informations, exécute :
    git config --global --list

Utilité

La configuration de Git permet de :

  • Identifier l'auteur de chaque commit.
  • Maintenir un historique clair des contributions.
  • Faciliter la collaboration sur des projets partagés.
